CopyLink2MD

  • CopyLink2MD is a simple tool to save the url of a website to a markdown-file. Copy a url with [hotkeycopyplus] to save it.
  • The application is especially helpful if you use markdown files to organize all your notes, with for example Obsidian, Zettlr, Typora, etcetera.
  • The app gives you the option to set a folder as the root for notes with bookmarks, e.g. (a subfolder of) your vault. Every time you want to save a url: select it and instead of Ctrl-c you press [hotkeycopyplus] (or other hotkey). It will show a menu to edit or add some more information (e.g. tags) or just press enter to save it in the default markdown-file.
  • You can automatically add tags for frequently used websites like youtube, github, imdb, etcetera. This makes it easy to get e.g. your favorite movie list. You can add or change the preferred websites for autotagging in the Settings menu.
  • Copy a symbolic link of cl2md.exe (or cl2md.ahk if you have autohotkey installed) to your startup-folder to run the app automatically at startup. See here how.
